# GoBarber API

## Overview

API para gerenciamento de agendamentos com profissionais por horários.
Inclui cadastro de usuário, cadastro de profissionais, cadastro de agendamentos e notificações.

## Features

- NodeJS
- Postgres
- Sequelize
- MongoDB
- Redis
- NodeMailer
- Sentry
- ESLint
- Prettier
- Nodemon
